           Лабораторная работа

          Методы поиска решений

1 Цель работы Изучение основных подходов к поиску решений

2 Методологические указания по организации самостоятельной работы
  Приступая к выполнению работы  студент должен повторить материалы
лекций разобраться в сути изучаемых методов ознакомиться с основными
понятиями и положениями

3 Описание лабораторной установки
  В качестве лабораторной установки используется ПЭВМ программы
реализующие рассматриваемые методы поиска решений

4 Порядок выполнения работы
 1)Изучите исходный текст программы реализующей метод поиска в
   глубину
 2)Выполните программу поиска в глубину и оцените полученное
   решение
 3)Изучите исходный текст программы реализующей метод поиска в
   ширину
 4)Выполните программу поиска в ширину и оцените полученное
   решение
 5)Изучите исходный текст программы реализующей эвристический
   метод поиска "восхождением на холм"
 6)Выполните программу поиска "восхождения на холм" и оцените
   полученное решение
 7)Изучите исходный текст программы реализующей эвристический
   метод поиска по минимальной стоимости
 8)Выполните программу поиска по минимальной стоимости и оцените
   полученное решение
 9)Изучите исходный текст программы поиска множественных решений
   по методу "с удалением пути"
10)Выполните программу поиска множественных решений по методу
  "с удалением пути" и оцените полученные решения
11)Изучите исходный текст программы поиска множественных решений
   по методу "с удалением узла"
12)Выполнить программу поиска множественных решений по методу
   "с удалением узла" и оцените полученные решения
13)Изучите исходный текст программы поиска оптимального решения
14)Выполните программу поиска оптимального решения и оцените
   метод поиска

                        5.Содержание отчета.
1.Цель работы
2.Схема контрольного примера.
3.По каждому рассмотренному методу кратко изложить суть, возможности,
  ограничения, полученное решение и его характеристику.
4.Выводы по работе.

                               Контрольные вопросы.
1)Что такое множественное решение ? Что оно позволяет? В каких случаях
следует использовать методы получения множественных решений?
2)Какие возможности и ограничения методов "слепого" поиска?
3)Что позволяет включение в метод эвристики? Возможно ли создать
универсальный эвристический метод?
4)Что подразумевается под поиском "оптимального решения"? В каком
смысле полученное решение можно считать оптимальным?
5)Как осуществить выбор метода поиска?
