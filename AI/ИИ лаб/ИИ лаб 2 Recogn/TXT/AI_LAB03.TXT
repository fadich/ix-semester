                       Лабораторная работа
                     -------------------------
                     
     	 		Распознавание образов
                       ------------------------	

1.Цель работы
----------------
Изучение некоторых подходов к распознаванию двумерных образов.Анализ
и сравнение методов распознавания по углу,по ключевым точкам и Дельта-Д раз-
познавателя.

2.Методические указания по организации самостоятельной работы.
---------------------------------------------------------------
	Приступая к выполнению работы,студент должен повторить материалы
лекции,изучить методы распознавания по углу,распознавания по ключевым точкам
и Дельта-Д распознаватель.Понять какие принципы заложены в каждом из них.
Обратить внимание на 5 критериев /вопросов/,по которым необходимо оценивать 
эти методы.

3.Описание лабораторной установки.
-----------------------------------
	В качестве лабораторной установки используется ПЭВМ,программа,реа-
лизующая методы распознавание по углу,распознавание по ключевым точкам и
Дельта-Д распознавание.

4.Порядок выполнения работы.
-----------------------------
   1.Изучить исходный текст программы распознавания по углу.
   2.Выполнить программу распознавания по углу при различных
взаиморасположениях треугольника и квадрата. Оценить этот метод,
ответив на 5 вопросов-критериев.
   3.Изучить исходный текст прграммы распознавания по ключевым точкам.
   4.Выполнить программу распознавания по ключевым точкам при различных
взаиморасполжениях треугольников и квадрата. Оценить этот метод, ответив
на 5 вопросов-критериев.
   5.Изучить исходный текст программы Дельта-Д распознавателя.
   6.Выполнить программу Дельта-Д распознавателя при различных взаиморасположе-
ниях треугольников и квадрата.Оценить этот метод,ответив на 5 вопросов-крите-
риев.
   7.Оформить отчет.

5.Содержание отчета.
----------------------
   1.Цель работы.
   2.Примеры для каждого метода распознавания и выводы по примерам.
   3.Ответы на вопросы-критерии по каждому методу.
   4.Выводы по работе.

Контрольные вопросы.
-----------------------
   1.В чем состоит метод распознавания по углу?
   2.Возможности и ограичения метода распознавния по углу.
   3.В чем состоит метод распознавания по ключевым точкам?
   4.Какие возможности и ограничения метода распознавания по ключевым точкам?
   5.В чем состоит метод Дельта-Д распознавания?
   6.Какие возможности и ограничения Дельта-Д распознавателя?
